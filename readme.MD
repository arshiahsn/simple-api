<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/arshiahsn/simple_api.git
   ```
2. Install Maven packages
   ```sh
   ./mvnw clean install
   ```
   
### Running


  ```sh
  ./mvnw run
  ```
  
### Usage
Open your browser and navigate to:

  ```sh
  localhost:8080/posts?
  ```
